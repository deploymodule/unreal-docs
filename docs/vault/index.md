# Vault

> Generic inventory that stores lean, hydrates on demand.

Vault is a generic inventory container system. Items are stored as minimal data structs and hydrated to full `UItemDefinition` objects only when needed. Supports slots, weight limits, modifier stacks, transaction logging, rule-based operations, and full replication.
