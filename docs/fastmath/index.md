# FastMath

High-performance Blueprint math node library — SIMD-accelerated math, curves, noise, and geometry.

| | |
|---|---|
| **Category** | Utility |
| **UE Version** | 5.5+ |
| **Blueprint API** | Yes |
| **C++ API** | Yes |
| **GAS Required** | No |

FastMath fills the gaps in UE5's built-in math library with production-quality implementations of math operations that Blueprint developers commonly reach for. All functions are implemented in native C++ with SIMD paths where beneficial. Categories include: advanced interpolation, curve evaluation, procedural noise, spatial geometry, quaternion utilities, and statistical math. Each node is named and described to be discoverable from Blueprint search without prior knowledge of its implementation.
