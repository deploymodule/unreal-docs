# Perpetual

> Animate anything without Sequencer or custom Blueprint.

Perpetual is a data-driven procedural animation component that handles doors, levers, switches, platforms, and any actor with configurable multi-stage sequences. Define stages in a Data Asset and attach the component — no Sequencer setup and no custom Blueprint event graphs required.
