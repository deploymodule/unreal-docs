# Reach

> Lightweight component-based interaction — zero forced base classes.

Reach is a minimal interaction system built on two components: a detector on the player and an interactable on each prop. It supports periodic sweep detection, optional replication, and four interaction modes (Instant, Hold, Smash, Rhythm) without requiring any base class changes to your actor or character.
