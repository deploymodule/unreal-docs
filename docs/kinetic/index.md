# Kinetic

> Curated animation graph nodes that do the hard work for you.

Kinetic ships a library of Animation Graph nodes for layered animation blending, procedural body dynamics, and state-machine simplification. Each node is data-driven via preset bone masks stored as Data Assets, letting animators iterate without touching C++.
