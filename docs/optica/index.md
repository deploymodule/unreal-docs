# Optica

Tactical bodycam/FPS C++ plugin — weapon physics, procedural camera motion, and sensor emulation.

| | |
|---|---|
| **Category** | Gameplay |
| **UE Version** | 5.5+ |
| **Blueprint API** | Yes |
| **C++ API** | Yes |
| **GAS Required** | No |
| **Replication** | Yes |

Optica delivers the complete camera and weapon feel stack for tactical FPS and bodycam-style games. It handles procedural weapon sway, spring-based recoil, ADS (aim down sights) transitions, procedural breathing, footstep camera bob, and sensor emulation (sensor noise, vignette, exposure lag, motion blur scaling). Every parameter is tunable at runtime, and the full system runs on the render thread for zero gameplay-thread latency.
