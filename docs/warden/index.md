# Warden

> 66 modular AI tasks out of the box.

Warden is a genre-agnostic AI StateTree task library shipping 66 modular tasks, 9 reusable conditions, and EQS query templates for movement, perception, combat, pet, and companion AI. A dual perception architecture cleanly separates memory (what the AI knows) from sensing (what it currently detects).
