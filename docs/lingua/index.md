# Lingua

Data asset-driven language and localization system for in-world languages and custom scripts.

| | |
|---|---|
| **Category** | Narrative |
| **UE Version** | 5.5+ |
| **Blueprint API** | Yes |
| **C++ API** | Yes |
| **GAS Required** | No |
| **Replication** | No (client-side) |

Lingua manages in-world fictional languages separately from UE's built-in localization system. It allows text to be written in a fictional language, then revealed progressively as the player gains proficiency — ideal for alien scripts, ancient runes, coded messages, or any game that features language as a game mechanic. Each language is defined in a Data Asset with its own glyph set, proficiency system, and optional real-world translation mapping.
