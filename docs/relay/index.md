# Relay

> High-performance gameplay message routing via GameplayTags.

Relay is a priority-aware, filter-capable gameplay message bus. Systems publish tagged messages; listeners subscribe with tag queries, priority filters, and optional sticky channels. Full Blueprint support and an async listen action make event-driven gameplay architecture clean and fast.
