# OmniSense

> AI perception that scales to 100+ agents without breaking a sweat.

OmniSense is an event-driven, spatially-partitioned AI sense system that replaces AIPerception. It provides frame-sliced vision checks, acoustic pings, scent trail tracking, and a team relationship matrix — all tuned to sustain 60fps with over a hundred active agents.
