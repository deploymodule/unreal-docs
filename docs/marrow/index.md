# Marrow

Curated C++ horror gameplay systems — sanity, dread, perception AI, and environmental horror.

| | |
|---|---|
| **Category** | Gameplay |
| **UE Version** | 5.5+ |
| **Blueprint API** | Yes |
| **C++ API** | Yes |
| **GAS Required** | No |
| **Replication** | Yes (where applicable) |

Marrow is a collection of tightly focused horror game systems that cover the mechanics most horror games need but UE5 doesn't provide out of the box. Each system is independently usable — you can use Sanity without Dread, or Perception without Sanity. Systems communicate via delegates so you can connect them however your game demands without hardcoded coupling.
