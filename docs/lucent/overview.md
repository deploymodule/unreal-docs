# Lucent — Overview

## Architecture

Lucent runs as a single view extension with each lens element as a separate compute dispatch.

```
ULucentSubsystem  (UEngineSubsystem)
    └── FLucentViewExtension  (FSceneViewExtensionBase)
            └── PrePostProcessPass_RenderThread()
                    ├── CoC.usf              — Circle of Confusion computation
                    ├── DOF_Near.usf         — Near field bokeh (tiled scatter)
                    ├── DOF_Far.usf          — Far field bokeh (gather)
                    ├── DOF_Composite.usf    — Near/far recombine
                    ├── Bloom.usf            — Multi-scale physical bloom
                    ├── Halation.usf         — Film-base halation glow
                    ├── Streaks.usf          — Anamorphic streak flares
                    ├── CA.usf               — Chromatic aberration (radial)
                    └── Distortion.usf       — Barrel/pincushion lens distortion
```

All passes share a single per-frame constant buffer. Disabled passes are fully skipped.

## Optical Elements

### Depth of Field

Lucent's DOF is a two-field (near + far) system. Near field uses a tiled-scatter approach — bright bokeh discs are scattered to a light accumulation buffer, then composited back. Far field uses a gather-based gather pass for smooth transitions into out-of-focus backgrounds.

The Circle of Confusion radius is computed from physical lens parameters:
- **Focal length** (mm)
- **Aperture** (f-stop)
- **Focus distance** (cm)
- **Sensor width** (mm)

Custom bokeh shapes can be authored as 128×128 grayscale textures (e.g., hexagonal, oval, star-shaped iris blades).

### Physical Bloom

A six-scale separable bloom derived from a Gaussian pyramid. Each scale can have independent color tint and intensity, enabling the characteristic "cyan core, orange halo" of anamorphic lenses.

### Halation

Identical to Tonal's halation but tuned for lens-based use (rather than film-base use). Spreads from near-white highlights into a soft red/orange corona.

### Anamorphic Streaks

Horizontal streak flares generated by anisotropic blur of above-threshold highlights. Streak length, color temperature, and flare count (1, 2, or 4 streaks) are all configurable.

### Chromatic Aberration

Radial chromatic aberration that grows from center to edge. Implemented as a per-channel UV offset with a configurable barrel profile. Axial CA (focus-dependent) is also supported.

### Barrel Distortion

A Brown-Conrady distortion model (k1, k2, k3 coefficients) for both barrel (wide angle) and pincushion (telephoto) lens profiles. The distortion is composited last to avoid artifacts from other passes.

## Key Concepts

| Concept | Detail |
|---------|--------|
| **Physical lens model** | DOF parameters map to real focal length, f-stop, and focus distance |
| **Sequencer integration** | All parameters are keyframeable via `ULucentSequencerTrack` |
| **Bokeh shape assets** | Custom iris shapes via `ULucentBokehAsset` (128×128 grayscale) |
| **Half-res DOF** | Near/far passes run at configurable resolution via `r.Lucent.DOF.ResScale` |
