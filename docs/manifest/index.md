# Manifest

> The definition store and asset resolver for your runtime world.

Manifest is a centralized subsystem for registering `PrimaryAssetId`s, resolving runtime objects by GUID, managing GUID pools, and optionally streaming assets through Courier. It is the single source of truth for "what things exist" in your game world at runtime.
