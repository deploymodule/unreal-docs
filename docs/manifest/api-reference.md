# Manifest — API Reference

## UManifestSubsystem

| Function | Signature | Description |
|---|---|---|
| `Resolve` | `void (FPrimaryAssetId, ECourierPriority, FOnManifestResolved)` | Resolves an asset by ID, loading it via Courier if necessary. |
| `ResolveSync` | `UManifestResolvedObject* (FPrimaryAssetId)` | Synchronously returns the resolved object if already loaded, else null. |
| `RegisterCatalog` | `void (UManifestCatalog*)` | Adds all entries from a catalog to the resolution map. |
| `UnregisterCatalog` | `void (UManifestCatalog*)` | Removes all entries from a catalog from the resolution map. |
| `GetAssetsByCategory` | `TArray<FPrimaryAssetId> (FName)` | Returns all registered asset IDs in a given category. |
| `GetAssetsByTag` | `TArray<FPrimaryAssetId> (FGameplayTag)` | Returns all registered asset IDs carrying a specific tag. |
| `IsRegistered` | `bool (FPrimaryAssetId)` | Returns true if the asset ID is currently in the resolution map. |
| `GetGuidPool` | `FManifestGuidPool& (FName Namespace)` | Returns (or creates) a GUID pool for the given namespace. |

## UManifestCatalog

| Property | Type | Description |
|---|---|---|
| `CatalogName` | `FName` | Human-readable name for this catalog. |
| `Entries` | `TArray<FManifestEntry>` | All asset definitions in this catalog. |

## FManifestEntry (Struct)

| Field | Type | Description |
|---|---|---|
| `AssetId` | `FPrimaryAssetId` | The primary asset identifier. |
| `DisplayName` | `FText` | Human-readable display name. |
| `Category` | `FName` | Logical category (e.g., `"Item"`, `"Character"`). |
| `Tags` | `FGameplayTagContainer` | Gameplay tags for filtering. |
| `SoftPath` | `FSoftObjectPath` | Soft reference to the actual asset. |

## UManifestResolvedObject

| Function | Description |
|---|---|
| `GetAssetId() const` | Returns the `FPrimaryAssetId` this object represents. |
| `GetObject() const` | Returns the loaded `UObject*` or null. |
| `IsLoaded() const` | Returns true if the underlying asset is fully loaded. |
| `GetLoadState() const` | Returns `ECourierLoadState` (Unloaded, Loading, Loaded, Failed). |

## FManifestGuidPool

| Function | Signature | Description |
|---|---|---|
| `GenerateGuid` | `FGuid ()` | Generates and records a new unique GUID in this pool. |
| `IsValidGuid` | `bool (const FGuid&)` | Returns true if the given GUID was generated by this pool. |
| `GetCount` | `int32 ()` | Returns the number of GUIDs currently in this pool. |

## Delegates

| Delegate | Parameters | Description |
|---|---|---|
| `FOnManifestResolved` | `UManifestResolvedObject*` | Called when asset resolution completes (success or failure). |
