# Gimbal

> Modular data-driven camera system for every perspective.

Gimbal provides a priority-stack camera architecture supporting third-person, first-person, top-down, side-scroller, fixed, and orbit modes — all configurable via Data Assets with no C++ required. Built-in lock-on targeting and seamless runtime blending make it the only camera plugin you need.
