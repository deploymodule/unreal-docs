# Sonance

> One audio framework for music, ambience, foley, and dialogue.

Sonance is a data-driven audio management framework supporting music stems, spatial ambience layers, foley banks, dialogue playlists, and sound prioritization — all built on Metasound. It ships with language/localization-aware dialogue routing so the right audio plays for the right locale without custom Blueprint.
