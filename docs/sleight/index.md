# Sleight

> Universal card game framework — deckbuilders, TCGs, and everything in between.

Sleight is a data-driven, presentation-agnostic card game framework. It handles decks, hands, draw/discard, effect stacks, and combat resolution without assuming any visual layer, making it compatible with 2D UMG, 3D card actors, or headless server logic.
