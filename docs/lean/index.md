# Lean

> 360-degree collision-aware leaning for cover and peek mechanics.

Lean provides a `ULeanComponent` that handles full 360-degree leaning with real-time collision checking, movement-limit enforcement when the lean path is obstructed, and optional Enhanced Input binding for instant cover-peek integration.
